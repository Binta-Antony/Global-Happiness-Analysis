
# 🌍 Global Happiness Analysis

Exploratory data analysis of the World Happiness dataset to identify key socio-economic factors influencing happiness levels across countries.

## 📊 Dataset
The analysis uses the **World Happiness Report dataset** combined with additional socio-economic indicators such as digital access and unemployment.

Features include:
- GDP per capita
- Social support
- Healthy life expectancy
- Freedom to make life choices
- Generosity
- Perceptions of corruption
- Digital access
- Unemployment rate

## ⚙️ Technologies Used
Python • Pandas • NumPy • Matplotlib • Jupyter Notebook

## 🔎 Project Workflow
Data Loading → Data Cleaning → Data Integration → Exploratory Data Analysis (EDA) → Correlation Analysis → Insights

## 📈 Key Insights
- Economic stability and social support show strong relationships with happiness levels.
- Digital access and unemployment also influence overall well-being.
- Countries with stronger socio-economic support systems tend to report higher happiness scores.

## 🎯 Goal
To understand how different socio-economic factors contribute to global happiness and identify patterns across countries.
